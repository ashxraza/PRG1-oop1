# Object-Oriented Programming (OOP)

Welcome! This activity is split into **6 short parts**. Each part lives in its own folder and has a small README, a working example, and a challenge for you to complete.

## Learning Objectives
By the end of this activity you will be able to:
- Define **Class**, **Object**, **Constructor**, **Method**, and **Attribute**
- Create multiple classes that use **inheritance**
- Create at least **ten objects** from your classes
- Describe the **Object-Oriented Programming (OOP)** paradigm
- Apply **encapsulation** using getters and setters

## How to work through this activity

Each numbered folder is a self-contained step. Work through them in order:

1. **`01-first-class/`** — Your first class: `Person`
2. **`02-encapsulation/`** — Getters, setters, and protecting your data
3. **`03-inheritance-staff/`** — Inheritance with `AdaStaff`
4. **`04-inheritance-student/`** — Inheritance with `AdaStudent`
5. **`05-composition-cohort/`** — Composition with a `Cohort` class
6. **`06-final-challenge/`** — Put it all together

Inside each folder you'll find:

| File | What to do with it |
|---|---|
| `README.md` | Read this first. It explains the part and lists the tasks. |
| `starter.py` | A working example. **Predict** what it does, then run it. |
| `challenge.py` | Your turn — complete the tasks here. |
| `solution.md` | Hidden answers. Only open after attempting the challenge! |

## Reference
- **`GLOSSARY.md`** — All the key terms (Class, Object, Method, etc.) in one place. Refer back to this any time.
- **`WHY_OOP.md`** — A short explanation of why OOP exists and what its core principles are. Read this at the end.

## Quick checklist
By the time you finish you should have:
- [ ] A `Person` class (the base class)
- [ ] An `AdaStaff` class (inherits from `Person`)
- [ ] An `AdaStudent` class (inherits from `Person`)
- [ ] A `Cohort` class (contains `AdaStudent` objects)
- [ ] At least **10 objects** created across the activity
- [ ] Used encapsulation, inheritance, and composition

Let's go! Start with `01-first-class/`.
